Project 1: Sudoku Management System

Project Description: The Sudoku Management System is a comprehensive platform designed for players to register, login, and enjoy Sudoku games. Key features include a leaderboard to track completion times, player questions with admin responses, and a subscription-based model for competitive gameplay.

Database Schema:

  • Admin Table: Fields - admin_id, admin_name, email, password
  • Player1 Table: Fields - player_id, player_name, email, password
  • Subscription Table: Fields - subc_id, player_name, subc_type, amount, game_type, end_date
  • Payment Table: Fields - payment_id, subc_id, player_name, pay_mode, date_time
  • Questions Table: Fields - question_id, player_name, email, question
  • Replies Table: Fields - reply_id, player_name, question_id, reply

Technologies Used: HTML, CSS, JavaScript, PHP, MySQL, PHPMyAdmin, XAMPP server

Github link:"https://github.com/sahanac2004/sudoku.git"

Project 2:Tic-Tac-Toe

Project Description: This is a comprehensive Tic-Tac-Toe game. The game allows two players to compete by taking turns placing their marks (X and O) on a 3x3 grid. The objective is to get three of their marks in a row (horizontally, vertically, or diagonally) to win the game.

Key features of the game include dynamic game board creation using JavaScript, player turn tracking and switching, win condition checking, draw condition checking, and a reset button to start a new game.

Technologies Used: HTML, CSS, JavaScript.
